# CV-Builder

This is an application that lets you build your own, personal CV. This app was made using Javascript, React JS, React-PDF (kendo-react) library and Material-UI.

| :--------------------------------------------------------------------------------------: | :---------------------------------------------------------------------------------: | :---------------------------------------------------------------------------------------: |

- React JS
  - Hooks
  - Context
  - Reducers
- Material-UI
